### 创建画布

| 事件名称       | 创建画布 |
| ---           | ---      |

| 输入参数 6     | 纵深：画布的显示优先级，数值越大优先级越高        |
| 输出参数       | 无        |
| 返回值         | 无        |
| 先决条件       | 无        |
| 调用函数       | 无        |

例：
```js
/* 这是UI编辑器中 创建画布 的参考例程 */

……
```
